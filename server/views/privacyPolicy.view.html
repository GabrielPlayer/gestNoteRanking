Welcome to GestNote Ranking! This Privacy Policy outlines how we collect, use, and protect user data. By using our extension, you agree to the terms outlined in this policy.

We use the collected information solely for the following purposes:

4. Information Sharing

We want to assure you that:

5. Security Measures

We prioritize the security of your data:

  • Anonymization: All collected data is anonymized, ensuring user privacy.
